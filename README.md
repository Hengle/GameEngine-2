# GameEngine

Simple engine for viewing 3d models
requirements:  
OpenGL 4.3  
OpenAL driver  
.Net Framework 4.8  

Features
* Open 3d model with one click
* Support for MMD pmx models, Pokemon Masters lmd models and bone animations
* Mesh, UV morphing
